Use Text2qti to handle Canvas Quiz, homework and exams

  1. Here is the Text2qti on Github.
  2. There are several examples on the Github page.
  3. Here are some more examples:
    • Sample Homework which includes
      • Five groups of problems. Each group contains 3-4 problems.
      • One picks randomly one or two problems from each group.
      • Ten points for each problem.
    • If you use vim or neovim, source the file comand_qti.vim where
      • the fold method is set to :set foldmethod=marker
      • The command <leader><leader> under normal mode will remove unnecessary {{{ from the fold method to generate file without .tex and then it will run text2qti the newly generated file (without .tex) to generate a file with .zip.
      • Now you can upload .zip file to Canvas.
      • These steps are shown by SampleHomework.md.tex –> SampleHomework.md –> SampleHomework.zip

Video processing and animation

  1. Use the Python package manim by 3 Blue 1 Brown to generate animation videos.
    • Here are some examples to walk through.
  2. The video file should be mmdd_S#.mp4, # be replaced by 5,6,7.
  3. cp 0917_S6_content.txt mmdd_S#_content.txt Edit the this content file for the section names and etc.
  4. cp SA-B.txt SA-B_S#.txt with A-B means Section A.B.
  5. Edit SA-B_S#.txt and run ./GenTitleVideo.sh SA-B_S# (without extension) to generate the title videos
  6. Finally, run ./splitvideo.lua mmdd_S# mmdd_S#_content.txt to see the command to be executed. If all looks good, run
$ splitvideo.lua mmdd_S# mmdd_S#_content.txt yes
  1. Here is the script splitvideo lua. This script depends on
    1. manim: used to generate the animation title videos.
    2. ffmpeg: used to split long course videos to short ones.
    3. MP4Box: used to combine animation video (without sound track with my course videos) ## Markdown related
